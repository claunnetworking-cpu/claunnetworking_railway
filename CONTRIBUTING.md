# Contribuindo

Obrigado por considerar contribuir!

## Como contribuir
1. Faça um fork do repositório
2. Crie uma branch: `git checkout -b feat/minha-feature`
3. Rode localmente:
   - `cd backend && npm i`
   - `cp ../.env.example .env` e ajuste se necessário
4. Faça commit com mensagem clara
5. Abra um Pull Request

## Padrões
- Prefira mudanças pequenas e bem descritas
- Inclua testes quando fizer sentido
- Não inclua segredos (tokens, chaves) no repositório
